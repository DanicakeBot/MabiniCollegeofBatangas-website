This project is a simple and responsive website for Mabini College of Batangas.
It is designed to provide information about the school, its programs, and allow students to easily access the enrollment form online.
📖 School information and programs (Junior High, Senior High, College)
🖼️ Custom banner and background design
📝 Direct "Enroll Now" button linked to Google Form
📱 Responsive layout (works on desktop and mobile)
🔗 Social sharing options
Live website https://danicakebot.github.io/mcb-website/
